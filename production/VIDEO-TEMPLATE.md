# Per‑Video Template

Copy this block for every video. It enforces the method: hook fast, promise something concrete, one big analogy, define every term before using it, end with a bridge.

```markdown
## [ID] · "[Title]"

- **Phase / Domain / Provider:**
- **Prerequisite video IDs:**
- **Length target:** 8–12 min

### The Promise
By the end you'll be able to ______ (concrete and testable).

### Hook (0–15s) — word for word
"..."

### The Big Analogy
The one everyday metaphor the whole video hangs on.

### Learning objectives (3–5)
- ...

### Beat sheet (with timestamps)
- 0:00 Hook + promise
- 0:30 1‑line recap of the prerequisite
- ... [mark where the ANALOGY, the DEMO, and the RECAP go]
- ~last Recap + bridge to next video

### Hands‑on / demo
- What to show on screen (free‑tier safe).
- **Cost‑safety note:** what to stop/delete; set a budget.

### New terms introduced (define each in one line)
- term — plain‑language definition
- ...

### Common misconceptions / FAQ (2–3)
- ...

### Recap + bridge
One‑line recap → teaser for the next video in the spine.

### CTA
Subscribe / comment prompt / next‑video nudge.
```

## Quality bar (check before publishing)

- [ ] No term is used before it's taught; every first use has an analogy **and** a definition.
- [ ] If it's a provider video, it explicitly references its Foundation ("remember subnets from `P1-NET-F`?").
- [ ] Provider order, when listing, is always **AWS → Azure → GCP → OCI → Alibaba**.
- [ ] Every hands‑on has a **free‑tier note** and a **cost‑safety note**.
- [ ] The hook lands in the first 15 seconds; the promise is concrete and testable.
- [ ] A total beginner could follow it without pausing to Google anything.
